
# DoD/Navy Financial Management Data Integration and Analysis Project

## Overview

This project demonstrates the integration and analysis of financial data relevant to DoD/Navy Financial Management using Power BI, Excel, and Python. It aims to provide meaningful insights through interactive dashboards, advanced charts, and automated data processing.

## Project Structure

### 1. Power BI and Excel Integration

- **Folder:** `PowerBI_and_Excel`
- **Description:** Contains Power BI reports (`*.pbix`) and Excel spreadsheets used for data transformation, visualization, and analysis.

### 2. Python Data Analysis and Automation

- **Folder:** `Python_Analysis`
- **Description:** Includes Python scripts and Jupyter notebooks (`*.ipynb`) for data preprocessing, statistical analysis, and automation tasks.

### 3. Documentation

- **File:** `Technical_Report.pdf`
- **Description:** Detailed technical report documenting project objectives, methodologies, tools used, data sources, analysis results, and insights derived.

### 4. Git Repository Management

- **Folder:** `.git`
- **Description:** Contains version control information and history of project changes.

## Usage Instructions

### Requirements

- Ensure Python (version X.X.X) and required libraries (NumPy, Pandas, SciPy) are installed.
- Microsoft Power BI Desktop and Excel (version X.X) should be installed for viewing and modifying Power BI reports and Excel sheets.

### Steps to Reproduce

1. Clone the Git repository to your local machine:
   ```
   git clone <repository-url>
   cd DoD_Navy_Financial_Management_Project
   ```

2. Open `PowerBI_and_Excel/DoD_Financial_Dashboard.pbix` using Power BI Desktop.
   - Customize and modify dashboards as needed for specific analysis requirements.
   - Connect to data sources or replace dummy data with actual data sources.

3. Use Excel files (`Excel_Data/Financial_Data.xlsx`) to explore detailed data analysis and create advanced charts.

4. Navigate to `Python_Analysis/` folder and run Jupyter notebooks (`*.ipynb`) to execute Python scripts for data preprocessing and analysis.
   ```
   jupyter notebook
   ```

5. Refer to `Technical_Report.pdf` for comprehensive insights and detailed methodology used in this project.
